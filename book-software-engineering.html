Software Engineering: A Practitioner's Approach

Roger S. Pressman, Bruce Maxim

Software Engineering is a comprehensive textbook that covers the principles and practices of software engineering. The book provides a balanced view of the software process, from initial customer communication through planning, modeling, construction, and deployment.

This book is essential for computer science and software engineering students, covering topics such as requirements engineering, design concepts, quality management, testing strategies, and project management. It includes real-world examples and case studies that help students understand how to apply software engineering principles in practice.
